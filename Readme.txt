Read this !!

1.First create a database named timecapsule in phpmyadmin console if you use MySql from xampp server

2.Import the timecapsule.sql file provided in this folder using the import tab of your database in phpmyadmin.

3.Load android project using Android Studio.

4.Move PhpAPI folder to your apache directory where php server is to be started.

5.Check for IP address of your machine and modify it in URLs class of android(if you are running in local network).

4.Now run the application in your mobile or emulator.

NOTE: Start MySql database before launching the application.
